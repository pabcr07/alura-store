<h2>Propósito del Análisis</h2>

El Sr. Juan necesita decidir en qué tienda expandir sus operaciones. Para ello, este análisis evalua y compara: 
<li>Ingresos totales acumulados</li>
<li>Distribución de ventas por categorías</li>
<li>Identificación de productos mas y menos vendidos</li>
<li>Promedio de calificaciones</li>
<li>Análisis de costos de envío</li>
